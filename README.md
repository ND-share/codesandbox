## Build Setup

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:2999
# Codesandboxを無理やりproductionで起動させるため
$ yarn d

# build for production and launch server
$ yarn start

# generate static project
$ yarn generate
```
